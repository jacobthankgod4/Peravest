```
❌ "Investor community"
❌ "Investment returns"
❌ "Guaranteed investment"
❌ "Investment risk"
❌ "Investment portfolio"
```

### Use Instead
```
✅ "Participate in real estate projects"
✅ "Participation opportunity"
✅ "Member community"
✅ "Member benefits"
✅ "Projected member benefits"
✅ "Participation risk"
✅ "Participation portfolio"
```

## Checklist for Content Review

When reviewing any content, check for:

- [ ] No "investment" terminology
- [ ] No "investor" references
- [ ] No "returns" or "ROI" language
- [ ] No "guaranteed" claims
- [ ] No "shares" or "securities" language
- [ ] Proper use of "member" instead of "investor"
- [ ] Proper use of "participation" instead of "investment"
- [ ] Proper use of "member benefits" instead of "returns"
- [ ] Proper use of "projected" before benefit claims
- [ ] Disclaimers present where needed
