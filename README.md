# PeraVest React Application

## Prerequisites
- Node.js 16+ and npm
- Download from: https://nodejs.org/

## Installation & Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Environment Variables
Configure in `.env`:
- `REACT_APP_API_URL` - Backend API URL
- `REACT_APP_PAYSTACK_PUBLIC_KEY` - Paystack public key
- `REACT_APP_APP_NAME` - Application name
- `REACT_APP_VERSION` - Version number

## Deployment
The `build/` folder contains production-ready files for deployment to any web server.
